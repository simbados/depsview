Dependency Viewer

Analyse Python dependencies from any public GitHub repository.

All processing happens in your browser. The only external calls are to api.github.com and pypi.org — no data is sent to any other server.

Your token is sent only to api.github.com and never transmitted to any other server.

Saved in your browser's local storage — persists between page visits and is readable by any JavaScript running on this page. Uncheck and submit to remove it.

The GitHub API allows 60 unauthenticated requests per hour. Provide a personal access token above to raise this to 5 000 req/hr and to access private repositories. Only the target directory and up to two levels of subdirectories are scanned for dependency files.
